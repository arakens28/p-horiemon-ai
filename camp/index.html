ABOUT

ホリエモンAI学校 1日Campとは？

チャット型AIだけでは「使えない」時代。エージェンティック AIの代表格 Claude Code を、
未経験から1日間の集中Campで実務レベルに引き上げる、唯一無二の短期プログラムです。

⚡

つきっきり1日間

朝9時〜夜18時の合計18時間。つきっきりで講師がサポート。動画講義のように途中で挫折することはありません。

🛠

Claude Code 実機操作

インストール・設定・GitHub連携・実際の業務適用まで、丸ごとハンズオン。受講後すぐ業務で使えます。

PROBLEM

こんな状況、ありませんか？

「AIには触れているはずなのに、最新の波には乗れていない」
その焦りを抱えている方が、いま急増しています。

😓

操作が難しそうに感じる

Claude Code、コードって聞いただけで構えてしまう。何から手をつければいいかすらわからない。

⚙️

インストール・設定で詰む

導入の最初のステップで挫折。ターミナルもGitHubも、自力で乗り越えられない。

💻

コーディング未経験で不安

「コード」と名前が付くだけで自分には無理だと諦めている。エンジニアじゃないと使えないと思っている。

🔗

GitHub連携で迷子になる

リポジトリ、コミット、プッシュ、プルリク…用語の海で溺れて、肝心の業務適用までたどり着けない。

📺

動画講義が続かない

30分の動画を週に何回も観るのは無理。1本も最後まで観れずに学習が止まったまま。

🏃

競合に置いていかれている

XやSNSではClaude Codeの話題ばかり。社内では「AI使え」と言われるが、自分自身が使えていない。

SOLUTION

1日Campが、すべて解決します。

丸1日、9時から18時まで本気を出すだけ。
乗り遅れた波を、確実に取り戻せる環境を用意しました。

FEATURES

1日Campが、選ばれる3つの理由

動画講義でも、独学でもない。Camp形式だからこその確かな成果。

01
CAMP FORMAT

朝9時〜夜18時、つきっきりの「逃げ場のない」18時間

動画講義は、誰でも続かない。代表の荒木自身が「これまで500本以上の講義動画を作ったが、自分は1本も観ていない」と語る通りです。だからこそ、つきっきりCamp形式。逃げ場がないからこそ、1日で確実に習得できます。

02
UNLIMITED RETAKE

理解するまで「何度でも」再受講できる安心感

継続サポートに入れば、同じカリキュラムには何度でも参加可能。サポートなしでも2回まで参加できます。3度目はアシスタント参加（謝礼あり）、4度目は講師参加（講師料あり）、5度目はFC加盟して自分でCampを開催できる、独自の段階的サポート制度です。

03
REAL RESULTS

自社で実証済み。全社員13名が Claude Code を使いこなす

ホリエモンAI学校自身が、社員13名全員にこのCampを受講させ、いまや全員がClaude Codeで業務を自動化しています。「自分たちが本当に変わった」体験を、そのまま受講者にお届けする再現性のあるプログラムです。

TARGET

3つの「乗り遅れた」を、解決します

個人、経営者、チーム。それぞれの「やってしまった」を、Camp形式で本気で取り戻す。

🙋

乗り遅れた「方」

  • Claude Codeの存在は知っているが触っていない
  • 操作・インストール・設定が難しそうで踏み出せない
  • コーディング・GitHubの知識がない
  • 1日で未経験から実務レベルに到達したい
👔

乗り遅れた「方々」（経営者）

  • ChatworkやChatGPTを少し触っただけで「分かった気」になっている
  • Claude Code・Cursor・エージェンティックAIを触ったことがない
  • 社内で「お前らもAI使え」と堂々と言える立場になりたい
  • 意思決定者として、自分が一番遅れていることに焦りがある
🏢

乗り遅れた「チーム」（法人）

  • 全社的にClaude Codeを導入したい
  • 会社ごと貸切での開催を希望している
  • 1日完結プラン・半日（4時間）導入プランを検討したい
  • 御社の状況に合わせて柔軟にカスタマイズしてほしい
